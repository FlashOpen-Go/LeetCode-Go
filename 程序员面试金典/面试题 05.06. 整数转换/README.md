# 面试题 05.06. 整数转换

## 题目描述
整数转换。编写一个函数，确定需要改变几个位才能将整数A转成整数B。

#### 示例1:
> 输入：A = 29 （或者0b11101）, B = 15（或者0b01111）
> 输出：2

#### 示例2:
> 输入：A = 1，B = 2
> 输出：2

#### 提示:
    1、A，B范围在[-2147483648, 2147483647]之间

## 解析
#### 方法1：模拟计算
- 移位依次比较两个数的每一位是否相同

## 代码实现
### CPP
#### 方法1：模拟计算
```C++
class Solution {
public:
    int convertInteger(int A, int B) {
        int ret = 0;
        for (int i = 0 ; i < 32; i++)
        {
            if ( (A & 0x01) != (B & 0x01))
                ret++;
            A >>= 1;
            B >>= 1;
        }
        return ret;
    }
};
```
